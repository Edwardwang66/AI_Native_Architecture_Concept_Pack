# AI-Native Architecture Concept Pack

This pack contains:

1. `AI_Native_Architecture_and_MCP_Product_Strategy_zh.md`  
   Chinese product strategy covering the AI architecture skill, MCP control plane, whitepaper/site, vertical reference products, go-to-market, risks, and a 90-day validation plan.

2. `ai-native-architect/SKILL.md`  
   A standards-oriented Agent Skill design draft. It is intentionally marked as unvalidated until tested on real architecture scenarios.

## Recommended reading order

1. Read the product strategy.
2. Review the Skill workflow and output contract.
3. Run the Skill manually with several real teams.
4. Record failure modes and revise the instructions.
5. Only then package it for public distribution or build the hosted MCP control plane.
